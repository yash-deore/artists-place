## Run on localhost:

1. Clone the repository.
2. Run `npm install`
3. Run `npx hardhat node` and then deploy the smart contracts `npx hardhat run scripts/deploy.js --network localhost`
4. Run `npm start` to run the project on http://localhost:3000/
